## 2. 이미지 처리 모듈

### 파일: `src/core/image/restoration.py`

### 2.1 주요 클래스 및 기능

| 클래스 | 기능 | 사용 모델 |
|--------|------|----------|
| `ImageRestorer` | 통합 이미지 복원 파이프라인 | - |
| `ImageAnalyzer` | 이미지 품질 분석, 손상 유형 감지 | - |
| `SuperResolution` | 저해상도 이미지 초해상도 변환 | Real-ESRGAN |
| `FaceEnhancer` | 얼굴 영역 선명화 및 향상 | GFPGAN, CodeFormer |
| `ImageInpainter` | 긁힘, 얼룩 등 손상 복원 | LaMa |
| `ImageColorizer` | 흑백 사진 컬러화 | DeOldify |
| `ImageCaptioner` | 이미지 자동 캡션 생성 | BLIP-2 |
| `SceneClassifier` | 장면/상황 분류 | CLIP |

### 2.2 필요한 입력 데이터
```
입력 형식: JPG, JPEG, PNG, BMP, TIFF
권장 해상도: 제한 없음 (저해상도도 처리 가능)
파일 크기: 제한 없음 (메모리에 따라)

예시 데이터:
- 오래된 가족 사진 (흑백/컬러)
- 손상된 사진 (긁힘, 얼룩, 찢어짐)
- 저해상도 스캔 이미지
```

### 2.3 CLI 실행 방법

```bash
# 기본 실행 (단일 이미지)
python -m src.core.image.restoration input_image.jpg -o output_restored.jpg

# 옵션 설명:
#   input           : 입력 이미지 경로 또는 디렉토리
#   -o, --output    : 출력 경로
#   --mode          : 처리 모드 (quality/balanced/fast)
#   --scale         : 업스케일 배율 (2 또는 4)
#   --no-colorize   : 컬러화 건너뛰기
#   --no-faces      : 얼굴 향상 건너뛰기
#   --device        : 디바이스 (cuda/cpu)

# 예시 1: 단일 이미지 복원 (기본 설정)
python -m src.core.image.restoration data/input/images/old_photo.jpg -o data/output/

# 예시 2: 배치 처리 (디렉토리 전체)
python -m src.core.image.restoration data/input/images/ -o data/output/restored/

# 예시 3: 고품질 모드 + 4배 업스케일
python -m src.core.image.restoration photo.jpg -o restored.jpg --mode quality --scale 4

# 예시 4: 컬러화 없이 복원만
python -m src.core.image.restoration photo.jpg -o restored.jpg --no-colorize

# 예시 5: CPU에서 실행
python -m src.core.image.restoration photo.jpg -o restored.jpg --device cpu
```

### 2.4 Python 코드에서 사용

```python
from src.core.image.restoration import ImageRestorer, ProcessingMode

# 1. 복원기 초기화
restorer = ImageRestorer({"device": "cuda"})

# 2. 모델 로드
restorer.load_models({
    "realesrgan": "models/realesr-general-x4v3.pth",
    "gfpgan": "models/GFPGANv1.4.pth"
})

# 3. 단일 이미지 처리
result = restorer.process(
    image_path="data/input/images/old_photo.jpg",
    output_path="data/output/restored_photo.jpg",
    mode=ProcessingMode.BALANCED,
    options={
        "scale": 2,                # 2배 또는 4배
        "colorize": True,          # 흑백 → 컬러
        "enhance_faces": True,     # 얼굴 향상
        "inpaint": True,           # 손상 복원
        "denoise": True,           # 노이즈 제거
        "color_correction": True,  # 색상 보정
        "generate_caption": True,  # 캡션 생성
        "classify_scene": True     # 장면 분류
    }
)

# 4. 결과 확인
print(f"처리 시간: {result.processing_time:.2f}초")
print(f"품질 향상: {result.quality_improvement:.1f}")
print(f"처리 단계: {result.processing_steps}")
print(f"자동 캡션: {result.auto_caption}")
print(f"장면 태그: {result.tags}")
print(f"감지된 얼굴: {len(result.face_regions)}개")

# 5. 배치 처리
import glob
image_paths = glob.glob("data/input/images/*.jpg")

results = restorer.batch_process(
    image_paths=image_paths,
    output_dir="data/output/restored/",
    mode=ProcessingMode.BALANCED,
    options={"scale": 2, "colorize": True}
)

print(f"처리 완료: {len(results)}개 이미지")
```

### 2.5 출력 결과물
```
출력 파일:
- 복원된 이미지 (JPG/PNG)

result 객체 속성:
- restored_image: 복원된 이미지 (numpy array)
- metadata: 이미지 메타데이터
- processing_steps: 적용된 처리 단계 목록
- quality_improvement: 품질 향상 점수
- face_regions: 감지된 얼굴 영역
- auto_caption: 자동 생성 캡션
- tags: 장면 분류 태그
```

## 3. 문서 처리 모듈

### 파일: `src/core/document/ocr.py`

### 3.1 주요 클래스 및 기능

| 클래스 | 기능 | 사용 모델 |
|--------|------|----------|
| `DocumentProcessor` | 통합 문서 처리 파이프라인 | - |
| `DocumentPreprocessor` | 기울기 보정, 노이즈 제거, 이진화 | OpenCV |
| `PrintedTextOCR` | 인쇄체 문자 인식 | EasyOCR, Tesseract |
| `HandwritingRecognizer` | 손글씨 문자 인식 | TrOCR |
| `LayoutAnalyzer` | 문서 레이아웃 분석 | LayoutLMv3 |
| `NamedEntityRecognizer` | 개체명 인식 (인명, 지명, 날짜) | KoBERT/KoELECTRA |
| `KeywordExtractor` | 키워드 추출 | TF-IDF |
| `DocumentClassifier` | 문서 유형 분류 | 규칙 기반 |

### 3.2 필요한 입력 데이터
```
입력 형식: JPG, JPEG, PNG, TIFF, PDF (이미지화된 문서)
권장 해상도: 300 DPI 이상
지원 언어: 한국어, 영어, 일본어, 중국어 (혼용 가능)

예시 데이터:
- 편지 (손글씨/인쇄)
- 공문서, 증명서
- 일기, 메모
- 신문 스크랩
```

### 3.3 CLI 실행 방법

```bash
# 기본 실행
python -m src.core.document.ocr input_document.jpg -o output_dir/

# 옵션 설명:
#   input           : 입력 이미지 경로 또는 디렉토리
#   -o, --output    : 출력 디렉토리
#   --script        : 필체 유형 (printed/handwritten)
#   --languages     : 인식 언어 (ko en ja zh)
#   --device        : 디바이스 (cuda/cpu)

# 예시 1: 인쇄체 문서 OCR
python -m src.core.document.ocr document.jpg -o results/ --script printed

# 예시 2: 손글씨 문서 OCR
python -m src.core.document.ocr letter.jpg -o results/ --script handwritten

# 예시 3: 한국어-영어 혼용 문서
python -m src.core.document.ocr mixed_doc.jpg -o results/ --languages ko en

# 예시 4: 배치 처리
python -m src.core.document.ocr data/input/documents/ -o data/output/ocr_results/

# 예시 5: 자동 감지 (필체 유형, 언어)
python -m src.core.document.ocr document.jpg -o results/
```

### 3.4 Python 코드에서 사용

```python
from src.core.document.ocr import DocumentProcessor, ScriptType

# 1. 프로세서 초기화
processor = DocumentProcessor({"device": "cuda"})

# 2. 모델 로드
processor.load_models()

# 3. 단일 문서 처리
result = processor.process(
    image_path="data/input/documents/letter.jpg",
    script_type=ScriptType.HANDWRITTEN_SEMI,  # 또는 PRINTED, None(자동)
    languages=["ko", "en"]
)

# 4. 결과 확인
print(f"인식된 텍스트:\n{result.full_text}")
print(f"\n문서 유형: {result.metadata.document_type.value}")
print(f"필체 유형: {result.metadata.script_type.value}")
print(f"감지된 언어: {[l.value for l in result.metadata.languages]}")
print(f"추정 날짜: {result.metadata.estimated_date}")
print(f"신뢰도: {result.confidence:.2%}")
print(f"처리 시간: {result.processing_time:.2f}초")

# 5. 키워드 확인
print(f"\n키워드: {result.metadata.keywords}")

# 6. 개체명 확인
print("\n개체명:")
for entity in result.metadata.entities:
    print(f"  [{entity.entity_type}] {entity.text}")

# 7. 텍스트 블록별 정보
print("\n텍스트 블록:")
for block in result.text_blocks[:5]:
    print(f"  [{block.block_type}] {block.text[:50]}...")

# 8. 배치 처리
import glob
doc_paths = glob.glob("data/input/documents/*.jpg")

results = processor.batch_process(
    image_paths=doc_paths,
    output_dir="data/output/ocr_results/"
)

# 9. 결과 저장 (자동으로 txt, json 파일 생성)
for result in results:
    print(f"처리 완료: {result.metadata.document_type.value}")
```

```
출력 파일 (output_dir/):

- {filename}.json    : 메타데이터 (문서유형, 키워드, 개체명 등)

JSON 구조:

  ],
```

## 4. 동영상 처리 모듈

### 파일: `src/core/video/processing.py`

### 4.1 주요 클래스 및 기능

| 클래스 | 기능 | 사용 모델/라이브러리 |
|--------|------|----------|
| `VideoProcessor` | 통합 비디오 처리 파이프라인 | - |

### 4.2 필요한 입력 데이터
```
입력 형식: MP4, AVI, MOV, MKV, WEBM
권장 해상도: 제한 없음
오디오: 있으면 자막 생성 가능

예시 데이터:
- 가정용 비디오 (VHS 디지털화)

```

### 4.3 CLI 실행 방법

```bash
# 기본 실행
python -m src.core.video.processing input_video.mp4 -o output_dir/

# 옵션 설명:
#   input             : 입력 비디오 경로 또는 디렉토리
#   -o, --output      : 출력 디렉토리 (필수)
#   --enhance         : 품질 향상 적용
#   --stabilize       : 손떨림 보정 적용
#   --color-correct   : 색상 보정 적용
#   --no-scenes       : 장면 분할 건너뛰기
#   --no-keyframes    : 키프레임 추출 건너뛰기
#   --no-audio        : 오디오 추출 건너뛰기
#   --no-subtitles    : 자막 생성 건너뛰기
#   --device          : 디바이스 (cuda/cpu)

# 예시 1: 기본 분석 (장면분할 + 키프레임 + 자막)
python -m src.core.video.processing family_video.mp4 -o results/

# 예시 2: 품질 향상 포함
python -m src.core.video.processing old_video.mp4 -o results/ --enhance

# 예시 3: 손떨림 보정 + 색상 보정
python -m src.core.video.processing shaky_video.mp4 -o results/ --stabilize --color-correct

# 예시 4: 자막만 생성 (다른 처리 건너뛰기)
python -m src.core.video.processing video.mp4 -o results/ --no-scenes --no-keyframes

# 예시 5: 전체 처리 (향상 + 안정화 + 색보정)
python -m src.core.video.processing video.mp4 -o results/ --enhance --stabilize --color-correct

# 예시 6: 배치 처리
python -m src.core.video.processing data/input/videos/ -o data/output/video_results/
```

### 5.1 주요 클래스 및 기능

## 7. 통합 파이프라인 실행

### 7.1 전체 자료 일괄 처리 스크립트

```python
"""
전체 자료 일괄 처리 스크립트
"""
import os
import glob
from pathlib import Path

# 모듈 임포트
from src.core.image.restoration import ImageRestorer
from src.core.document.ocr import DocumentProcessor
from src.core.video.processing import VideoProcessor
from src.core.audio.transcription import AudioProcessor
from src.core.oral_history.processing import OralHistoryProcessor

def process_all_materials(input_dir: str, output_dir: str, device: str = "cuda"):
    """전체 자료 일괄 처리"""
    
    # 출력 디렉토리 생성
    os.makedirs(output_dir, exist_ok=True)
    
    # ========== 1. 이미지 처리 ==========
    print("\n" + "="*50)
    print("1. 이미지 처리")
    print("="*50)
    
    image_extensions = {'.jpg', '.jpeg', '.png', '.tiff', '.tif', '.bmp'}
    image_paths = [
        p for p in Path(input_dir).rglob("*")
        if p.suffix.lower() in image_extensions
    ]
    
    if image_paths:
        image_restorer = ImageRestorer({"device": device})
        image_restorer.load_models()
        
        results = image_restorer.batch_process(
            image_paths=[str(p) for p in image_paths],
            output_dir=os.path.join(output_dir, "images"),
            options={"scale": 2, "colorize": True, "enhance_faces": True}
        )
        print(f"이미지 처리 완료: {len(results)}개")
    
    # ========== 2. 문서 처리 ==========
    print("\n" + "="*50)
    print("2. 문서 처리")
    print("="*50)
    
    # 문서 이미지 (별도 폴더에 있다고 가정)
    doc_dir = os.path.join(input_dir, "documents")
    if os.path.exists(doc_dir):
        doc_processor = DocumentProcessor({"device": device})
        doc_processor.load_models()
        
        doc_paths = glob.glob(os.path.join(doc_dir, "*.jpg")) + \
                    glob.glob(os.path.join(doc_dir, "*.png"))
        
        results = doc_processor.batch_process(
            image_paths=doc_paths,
            output_dir=os.path.join(output_dir, "documents")
        )
        print(f"문서 처리 완료: {len(results)}개")
    
    # ========== 3. 동영상 처리 ==========
    print("\n" + "="*50)
    print("3. 동영상 처리")
    print("="*50)
    
    video_extensions = {'.mp4', '.avi', '.mov', '.mkv', '.webm'}
    video_paths = [
        p for p in Path(input_dir).rglob("*")
        if p.suffix.lower() in video_extensions
    ]
    
    if video_paths:
        video_processor = VideoProcessor({"device": device})
        video_processor.load_models()
        
        results = video_processor.batch_process(
            video_paths=[str(p) for p in video_paths],
            output_dir=os.path.join(output_dir, "videos"),
            options={
                "detect_scenes": True,
                "extract_keyframes": True,
                "generate_subtitles": True
            }
        )
        print(f"동영상 처리 완료: {len(results)}개")
    
    # ========== 4. 음성 처리 ==========
    print("\n" + "="*50)
    print("4. 음성 처리")
    print("="*50)
    
    audio_extensions = {'.wav', '.mp3', '.flac', '.m4a', '.ogg'}
    audio_paths = [
        p for p in Path(input_dir).rglob("*")
        if p.suffix.lower() in audio_extensions
    ]
    
    if audio_paths:
        audio_processor = AudioProcessor({
            "device": device,
            "whisper_model": "large-v3"
        })
        audio_processor.load_models()
        
        results = audio_processor.batch_process(
            audio_paths=[str(p) for p in audio_paths],
            output_dir=os.path.join(output_dir, "audio"),
            language="ko",
            diarize=True
        )
        print(f"음성 처리 완료: {len(results)}개")
    
    # ========== 5. 구술 인터뷰 심층 분석 ==========
    print("\n" + "="*50)
    print("5. 구술 인터뷰 심층 분석")
    print("="*50)
    
    # 구술 인터뷰 (별도 폴더)
    oral_dir = os.path.join(input_dir, "oral_history")
    if os.path.exists(oral_dir):
        oral_processor = OralHistoryProcessor({"device": device})
        oral_processor.load_models()
        
        oral_paths = glob.glob(os.path.join(oral_dir, "*.wav")) + \
                     glob.glob(os.path.join(oral_dir, "*.mp3"))
        
        for audio_path in oral_paths:
            base_name = os.path.splitext(os.path.basename(audio_path))[0]
            analysis = oral_processor.process(
                audio_path=audio_path,
                language="ko"
            )
            oral_processor.save_analysis(
                analysis,
                os.path.join(output_dir, "oral_history", base_name)
            )
        
        print(f"구술 인터뷰 분석 완료: {len(oral_paths)}개")
    
    print("\n" + "="*50)
    print("전체 처리 완료!")
    print("="*50)


if __name__ == "__main__":
    import argparse
    
    parser = argparse.ArgumentParser(description="전체 자료 일괄 처리")
    parser.add_argument("input_dir", help="입력 디렉토리")
    parser.add_argument("output_dir", help="출력 디렉토리")
    parser.add_argument("--device", default="cuda", help="디바이스")
    
    args = parser.parse_args()
    
    process_all_materials(args.input_dir, args.output_dir, args.device)
```
