## Unity AR/VR 구현 방향 (Unity를 활용한 AR/VR 구현에 대해 설명)

### 🎯 구현 범위 제안

Unity AR/VR은 **설계 및 프로토타입** 수준으로 제시하는 것이 현실적입니다. 이유:

1. **Unity는 C# 기반** - Python 시스템과 별도 프로젝트
2. **빌드/배포 환경 필요** - Android/iOS/VR 헤드셋
3. **3D 에셋 제작 필요** - 전시 공간, UI 등

### 📁 제공 가능한 산출물

| 산출물 | 내용 | 형식 |
|--------|------|------|
| **1. 시스템 설계서** | AR/VR 아키텍처, 기능 명세 | Markdown |
| **2. Unity C# 스크립트** | 핵심 기능 코드 | .cs 파일 |
| **3. API 연동 명세** | Python 서버 ↔ Unity 통신 | JSON Schema |
| **4. 씬 구성 가이드** | Unity 프로젝트 설정 방법 | 문서 |
