## 2. 디지털 아카이브 API 서버

### 파일: `src/core/api/server.py`

### 2.1 주요 클래스 및 기능

| 클래스 | 기능 |
|--------|------|
| `DigitalArchiveAPI` | API 서버 메인 클래스 |
| `ItemCreate` | 자료 생성 요청 모델 |
| `ItemResponse` | 자료 응답 모델 |
| `SearchRequest` | 검색 요청 모델 |
| `GraphResponse` | 그래프 응답 모델 |

### 2.2 API 엔드포인트 목록

| 카테고리 | 엔드포인트 | 메서드 | 설명 |
|---------|-----------|--------|------|
| **기본** | `/` | GET | API 루트 |
| | `/health` | GET | 헬스 체크 |
| | `/stats` | GET | 전체 통계 |
| **자료** | `/items` | GET | 자료 목록 |
| | `/items/{id}` | GET | 자료 상세 |
| | `/items` | POST | 자료 생성 |
| | `/items/{id}` | PUT | 자료 수정 |
| | `/items/{id}` | DELETE | 자료 삭제 |
| | `/items/{id}/upload` | POST | 파일 업로드 |
| **검색** | `/search` | POST | 전문 검색 |
| | `/search/suggestions` | GET | 자동완성 |
| **인물** | `/persons` | GET | 인물 목록 |
| | `/persons/{id}` | GET | 인물 상세 |
| | `/persons/{id}/items` | GET | 인물 관련 자료 |
| **장소** | `/locations` | GET | 장소 목록 |
| | `/locations/{id}/items` | GET | 장소 관련 자료 |
| **타임라인** | `/timeline` | GET | 연도별 타임라인 |
| **그래프** | `/graph` | GET | 관계 그래프 |
| | `/graph/related/{id}` | GET | 관련 자료 추천 |
| **처리** | `/items/{id}/process` | POST | AI 처리 요청 |
| | `/items/{id}/process/status` | GET | 처리 상태 |

### 2.3 서버 실행

```bash
# 방법 1: 직접 실행
python src/core/api/server.py --host 0.0.0.0 --port 8000

# 방법 2: uvicorn으로 실행
uvicorn src.core.api.server:create_api_server --host 0.0.0.0 --port 8000

# 방법 3: 자동 리로드 (개발용)
uvicorn src.core.api.server:create_api_server --host 0.0.0.0 --port 8000 --reload

# 방법 4: 워커 여러 개 (프로덕션)
uvicorn src.core.api.server:create_api_server --host 0.0.0.0 --port 8000 --workers 4
```

### 3.1 주요 클래스

| 클래스 | 기능 |
|--------|------|

### 3.2 스토리 유형

```python
class StoryType(str, Enum):
    FAMILY_HISTORY = "family_history"       # 가족사
    MIGRATION_JOURNEY = "migration_journey" # 이주 여정
    LIFE_CHRONICLE = "life_chronicle"       # 생애 연대기
    THEMATIC = "thematic"                   # 주제별
    PLACE_BASED = "place_based"            # 장소 중심
    INTERACTIVE = "interactive"             # 분기형

class NarrativeStyle(str, Enum):
    DOCUMENTARY = "documentary"     # 다큐멘터리 (객관적)
    PERSONAL = "personal"           # 개인적 (1인칭)
    STORYTELLING = "storytelling"   # 이야기체 (3인칭)
    POETIC = "poetic"              # 시적/감성적
    EDUCATIONAL = "educational"     # 교육적
```

```javascript
const BASE_URL = 'http://localhost:8000';

// 자료 목록 조회
async function getItems(page = 1, pageSize = 20) {
    const response = await fetch(
        `${BASE_URL}/items?page=${page}&page_size=${pageSize}`
    );
    return await response.json();
}

// 검색
async function searchItems(query, filters = {}) {
    const response = await fetch(`${BASE_URL}/search`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            query: query,
            ...filters,
            page: 1,
            page_size: 20
        })
    });
    return await response.json();
}

// 자료 생성
async function createItem(itemData) {
    const response = await fetch(`${BASE_URL}/items`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(itemData)
    });
    return await response.json();
}

// 관계 그래프 조회
async function getGraph(centerId, depth = 2) {
    const response = await fetch(
        `${BASE_URL}/graph?center_id=${centerId}&depth=${depth}`
    );
    return await response.json();
}

// 스토리 생성
async function generateFamilyStory(familyName, itemIds) {
    const response = await fetch(`${BASE_URL}/stories/generate/family`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            family_name: familyName,
            item_ids: itemIds
        })
    });
    return await response.json();
}

// 사용 예시
async function main() {
    // 자료 목록
    const items = await getItems();
    console.log(`총 ${items.total}개 자료`);
    
    // 검색
    const results = await searchItems('가족');
    console.log(`검색 결과: ${results.total}건`);
    
    // 스토리 생성
    const story = await generateFamilyStory('김씨 가족', ['item_001']);
    console.log(`스토리: ${story.title}`);
}

main();
```
