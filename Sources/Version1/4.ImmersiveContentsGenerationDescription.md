## 4단계: 실감형 콘텐츠 생성 계층

|구성요소 | 기능 | 기술 스택 |
|---------|------|----------|
|디지털 아카이브 | 웹 기반 자료 탐색, 고해상도 이미지 뷰어, 메타데이터 표시/편집, 관계 그래프 시각화, 타임라인 뷰, 검색/필터 | React + FastAPI | 
|AR 증강 뷰어 | 실물 사진 인식, 정보 오버레이, 인물 태그 표시, 관련 자료 연결, 음성 설명 | Unity AR Foundation | 
|VR 가상 전시 | 3D 가상 전시 공간, 자동 자료 배치, 음성 가이드, 자유 탐색/텔레포트 | Unity XR Toolkit | 
|인터랙티브 스토리 | AI 내러티브 생성, 멀티미디어 통합, 분기형 스토리, 구술 인용 삽입 | React + GPT-4 |

### 구현 범위에서 핵심적으로 구현해야 할 부분

#### 구현 우선순위 제안
✅ 1순위: 디지털 아카이브 (웹 API + 프론트엔드)

- FastAPI 백엔드: REST API 서버
- React 프론트엔드: 기본 웹 인터페이스 (선택)

⚠️ 2순위: 인터랙티브 스토리 생성

- AI 내러티브 생성기: GPT 기반 스토리 자동 생성

❌ 3순위: AR/VR (Unity 필요 - 별도 프로젝트)

- Unity 기반으로 별도 구현 필요
- 논문에서는 설계/프로토타입만 제시

#### 1. 디지털 아카이브 API (`server.py`)

|엔드포인트 | 메서드 | 기능 |
|---------|------|----------|
|`/items` | GET | 자료 목록 조회|
|`/items/{id}` | GET | 자료 상세 조회|
|`/items` | POST | 자료 생성|
|`/items/{id}` | PUT | 자료 수정|
|`/items/{id}/upload` | POST | 파일 업로드|
|`/search` | POST | 전문 검색|
|`/search/suggestions` | GET | 자동완성|
|`/persons` | GET | 인물 목록|
|`/persons/{id}/items` | GET | 인물 관련 자료|
|`/locations` | GET | 장소 목록|
|`/timeline` | GET | 타임라인 조회|
|`/graph` | GET | 관계 그래프|
|`/graph/related/{id}` | GET | 관련 자료 추천|
|`/stats` | GET | 전체 통계|


#### 2. 인터랙티브 스토리 생성기 (`generator.py`)

|클래스 | 기능|
|---------|----------|
|`StoryGenerator` | 스토리 생성 메인 엔진|
|`Story` | 스토리 데이터 모델|
|`StorySegment` | 스토리 세그먼트 (장면)|
|`LLMClient` | OpenAI/GPT 연동|
|`StoryExporter` | HTML/Markdown 내보내기|


|스토리 유형 | 설명|
|---------|----------|
|`family_history` | 가족사 스토리|
|`migration_journey` | 이주 여정 스토리|
|`life_chronicle` | 생애 연대기|
|`thematic` | 주제별 (결혼, 졸업 등)|
|`interactive` | 분기형 인터랙티브|


|스토리 API | 기능|
|---------|----------|
|`/stories/generate/family` | 가족사 생성|
|`/stories/generate/migration` | 이주 여정 생성|
|`/stories/generate/timeline` | 타임라인 생성|
|`/stories/generate/interactive` | 분기형 생성|
|`/stories/{id}/export/html` | HTML 내보내기|
|`/stories/{id}/export/markdown` | Markdown 내보내기|
